---
description: how to build the application to an installer
---
To build the application and generate installers for Windows (WiX and Squirrel), follow these steps:

1. Open your terminal in the project root directory.
2. Run the following command:
// turbo
```powershell
npm run make
```
3. Once the process completes, you can find the installers in the `out/make` directory.
